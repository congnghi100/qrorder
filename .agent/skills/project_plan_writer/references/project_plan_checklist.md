# 📅 Project Plan — Checklist & Templates

| Phase | Mốc thời gian | Deliverables chính |
|-------|--------------|-------------------|
| Kickoff | Tuần 1 | Assessment, Stakeholder List |

| Review & Approve | Tuần 4 | BRD/PRD Approved, Sign-off |
| Handoff | Tuần 4-5 | Sprint Backlog, API Spec, Test Cases |

## 2. Estimation Techniques

| Kỹ thuật | Khi nào dùng | Độ chính xác |
|---------|-------------|-------------|
| T-shirt sizing (S/M/L/XL) | Giai đoạn sớm, scope chưa rõ | ±50% |
| Story Points (Fibonacci) | Sprint planning, backlog grooming | ±30% |
| Giờ công (hours) | Task đã rõ, giai đoạn cuối | ±15% |
| 3-Point (O+4M+P)/6 | Khi cần confidence interval | ±20% |
